This project contains a comprehensive analysis of the NBA2k20 dataset with the help of Data Visualization technniques.
The following tasks were completed:
	1.) Draw a Histogram of the player's ratings
	2.) Draw a Box and Whiskers plot of the player's ratings by position.
	3.) Draw a Pie Chart of the player's positions.
	4.) Draw a Histogram of the draft year of the players.
	5.) Draw a Pie Chart for the draft round of the players.
	6.) Draw a Histogram and Pie Chart for the heigth attribute of each player.
	7.) Draw a Pie Chart of the player's nationalities.
	8.) Draw a Pie Chart of the player's salaries.
	9.) Draw a Histogram of the player's ages.
